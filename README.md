# LED Blinking using Raspberry Pi 4

## 📌 Overview
This project demonstrates how to blink an LED using the GPIO pins of Raspberry Pi 4 with Python.

## 🛠 Components Used
- Raspberry Pi 4 Model B
- LED
- 220Ω Resistor
- Breadboard
- Jumper Wires

## 💡 Working Principle
The Raspberry Pi sends HIGH and LOW signals through a GPIO pin to turn the LED ON and OFF repeatedly, creating a blinking effect.
